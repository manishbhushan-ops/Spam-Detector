# 📧 Spam Email Detector

A Machine Learning web application that detects whether a message is **Spam** or **Not Spam** using **Logistic Regression** and **TF-IDF Vectorization**.

Built with:
- Python
- Scikit-learn
- Streamlit
- NLP (Natural Language Processing)

---

# 🚀 Features

✅ Real-time spam detection  
✅ TF-IDF text vectorization  
✅ Logistic Regression model  
✅ Confidence score prediction  
✅ Clean and interactive Streamlit UI  
✅ Real SMS Spam Collection Dataset  

---

# 🧠 Machine Learning Concepts Used

- Text Cleaning
- TF-IDF Vectorization
- Logistic Regression
- NLP Basics
- Classification
- Probability Prediction
- Model Evaluation

---

# 📊 Dataset

Dataset Used:
**SMS Spam Collection Dataset**
